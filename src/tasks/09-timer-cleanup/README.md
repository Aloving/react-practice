# Задача 09 — Таймер с очисткой в useEffect

**Уровень:** средний

## Цель

Компонент с `setInterval` (таймер обратного отсчёта или секундомер) и корректная очистка интервала в cleanup `useEffect`.

## Требования

- Таймер: обратный отсчёт от N секунд до 0 или секундомер (0, 1, 2, …).
- Запуск по кнопке «Старт», при необходимости «Стоп» и «Сброс».
- Использовать `setInterval` внутри `useEffect`; в cleanup функции эффекта вызывать `clearInterval`, чтобы при размонтировании или смене зависимостей интервал не продолжал работать.
- Не допускать утечек: при размонтировании компонента таймер должен быть остановлен.

## Подсказки

- `useEffect(() => { const id = setInterval(() => { ... }, 1000); return () => clearInterval(id); }, [deps])`
- Зависимости: например, флаг «запущен» или значение счётчика — в зависимости от логики.

## Критерии приёмки

- Таймер идёт (или обратный отсчёт уменьшается) при нажатии Старт.
- Очистка интервала при размонтировании (проверить в React DevTools или при переключении задачи).
- Нет лишних интервалов при повторном запуске (один активный интервал).
