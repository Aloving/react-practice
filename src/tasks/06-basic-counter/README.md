# Задача 06 — Счётчик с useState

**Уровень:** базовый

## Цель

Реализовать простой счётчик: кнопки +1 и -1, отображение текущего значения. Базовая работа с `useState`.

## Требования

- Одна переменная состояния — число (значение счётчика).
- Две кнопки: увеличить на 1, уменьшить на 1.
- Текущее значение отображается на экране.
- Не выходить за разумные пределы (опционально: ограничить диапазон или не ограничивать — на твоё усмотрение).

## Подсказки

- `const [count, setCount] = useState(0)`
- В обработчиках вызывать `setCount(c => c + 1)` и `setCount(c => c - 1)`.

## Критерии приёмки

- Клик по кнопке +1 увеличивает значение на 1.
- Клик по кнопке -1 уменьшает значение на 1.
- Значение отображается корректно после каждого клика.
