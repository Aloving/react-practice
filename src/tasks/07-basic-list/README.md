# Задача 07 — Список с добавлением и удалением

**Уровень:** базовый

## Цель

Форма для добавления элемента в список и кнопка удаления у каждого элемента. Управление массивом в state.

## Требования

- State — массив элементов (например, строк или объектов с id и текстом).
- Поле ввода и кнопка «Добавить»: по нажатию новый элемент добавляется в список.
- У каждого элемента списка — кнопка удаления; по нажатию элемент удаляется из списка.
- Список отображается ниже формы.

## Подсказки

- `const [items, setItems] = useState<string[]>([])` или массив объектов с уникальным id.
- Добавление: `setItems(prev => [...prev, newItem])`.
- Удаление: `setItems(prev => prev.filter((_, i) => i !== index))` или по id.

## Критерии приёмки

- Можно добавить несколько элементов; они отображаются в списке.
- Кнопка удаления у элемента убирает только этот элемент.
- После перезагрузки страницы список сбрасывается (если не реализовано сохранение).
